# tablaDimayor - Gestor de Liga de Fútbol

Este programa de escritorio en Java sirve para administrar partidos y calcular las posiciones de la liga de fútbol Dimayor. Fue desarrollado para la materia de **Estructuras de Datos / Análisis de Sistemas** en el programa de **Técnico en Servicios de Seguridad Informática**.

## Funciones del programa
* **Cálculo automático**: Procesa los resultados de los partidos para actualizar los puntos y goles de cada equipo.
* **Estructura organizada**: Utiliza el patrón Modelo-Vista-Controlador (MVC) para separar el código de interfaz del código lógico.
* **Generación de reportes**: Exporta la tabla de posiciones final a un archivo HTML independiente (`Reporte_Dimayor.html`).

## Archivos del repositorio
* `src/`: Carpeta con los archivos de código fuente en Java divididos por modelo, vista y controlador.
* `Reporte_Dimayor.html`: Ejemplo del reporte final que genera el sistema.
* `docs/`: Documentación académica del informe entregable de la actividad.
